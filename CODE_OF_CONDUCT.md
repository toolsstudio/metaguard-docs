# Code of Conduct

## Our Commitment

This project is a documentation repository for a professional developer tool. Contributors and maintainers are expected to engage constructively and professionally.

---

## Standards

**Expected behavior:**
- Respectful, constructive communication
- Factual, evidence-based feedback on documentation
- Welcoming of corrections and improvements regardless of contributor experience level

**Unacceptable behavior:**
- Harassment or personal attacks
- Dismissive or condescending responses
- Off-topic, promotional, or spam content

---

## Scope

This Code of Conduct applies to all activity in this repository — issues, pull requests, discussions, and any other contribution channels.

---

## Enforcement

Violations may result in comments being removed, contributions rejected, or contributors blocked from the repository at the maintainers' discretion.

To report a concern, contact Tools Studio via [Discord](https://discord.gg/QNSJZGvRYM) or by opening a private GitHub issue.
