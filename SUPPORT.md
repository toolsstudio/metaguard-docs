# Support

**MetaGuard v1.0.0** | Publisher: Tools Studio | Free — MIT License

---

## Before Asking

Check [docs/troubleshooting.md](docs/troubleshooting.md) first. It covers the most common scan, apply, rollback, and cache issues with step-by-step resolutions.

---

## Community Support

**[Discord](https://discord.gg/rYbZZz5GH4)** is the primary support channel.

When posting, include:
- Unity version
- MetaGuard version (shown in `CHANGELOG.md`)
- Operating system
- Description of the issue
- Relevant Unity Console output (if applicable)

---

## Bug Reports

Open an issue on [GitHub](https://github.com/Afterix-Hub/MetaGuard/issues) for reproducible bugs not covered by the troubleshooting guide.

Use the `bug_report` issue template. Include a minimal reproduction case if possible.

---

## Documentation Issues

If documentation is incorrect, missing, or unclear, open an issue using the `documentation_request` template on [GitHub](https://github.com/Afterix-Hub/MetaGuard/issues).

Documentation contributions via pull request are welcome — see [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines.
