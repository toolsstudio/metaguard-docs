# Security Policy

## Scope

MetaGuard is a Unity Editor tool. It reads and writes `.meta` files and YAML asset data within the local Unity project directory. It has no network access, makes no external connections, and has no runtime component.

Security reports for this **documentation repository** are generally not applicable.

If you have identified a security concern in the MetaGuard tool itself, follow the disclosure process below.

---

## Supported Versions

| Version | Supported |
|---|---|
| 2.0.0 | ✅ |
| 1.0.0 | ❌ End of life |
