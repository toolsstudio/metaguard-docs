# Security Policy

## Scope

MetaGuard is a Unity Editor tool. It reads and writes `.meta` files and YAML asset data within the local Unity project directory. It has no network access, makes no external connections, and has no runtime component.

Security reports for this **documentation repository** are generally not applicable.

If you have identified a security concern in the MetaGuard tool itself, follow the disclosure process below.

---

## Supported Versions

| Version | Supported |
|---|---|
| 1.0.0 | ✅ |

---

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report security concerns directly to Tools Studio via [Discord](https://discord.gg/QNSJZGvRYM) (direct message to a maintainer) or via the GitHub repository's private security advisory feature.

Include:
- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Unity version and OS if relevant

You will receive an acknowledgment within 5 business days. If a fix is warranted, a patched release will be issued and credited to the reporter upon request.
